# PANSITO Game Documentation

## Project Overview
PANSITO is an engaging game designed to challenge players' strategic thinking and quick decision-making skills. This document serves as a comprehensive guide for understanding the game's mechanics, rules, objectives, and overall structure.

## Game Description
PANSITO is a multi-level puzzle game where players navigate through a series of challenges that increase in difficulty. The objective is to solve each puzzle within a time limit to advance to the next level.

## Gameplay Mechanics
- **Levels:** The game consists of several levels, each with unique challenges.
- **Timer:** Players must complete each level within a specific timeframe.
- **Scoring System:** Points are awarded based on the speed and accuracy of the solutions.

## Rules
1. Players must solve the puzzle to progress to the next level.
2. Each level has a different set of rules and mechanics.
3. Players can earn bonus points for completing levels faster than the designated time.

## Objectives
- Complete all levels.
- Achieve the highest score possible.
- Improve problem-solving and strategic thinking skills.

## Conclusion
PANSITO is not only a fun game but also a great way to hone your mental skills. Enjoy playing and good luck reaching the highest score!
